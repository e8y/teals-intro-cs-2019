What are blocks?

What do they do?

  • Perform some action
  • Produce a value

Classifying Blocks

  • Command blocks — return nothing, just perform some action
  • Reporter blocks — return a value, maybe after doing some calculation
  • Predicate blocks — returns a Boolean value, can be used inside conditional blocks like if

Custom Reporters & Predicates

You've already been using reporters and predicates...

But you can also make your own custom reporter and predicate blocks!

The report block

Exits your block and returns a value

Using this block makes your custom block into a reporter! (or predicate if returning a Boolean)
